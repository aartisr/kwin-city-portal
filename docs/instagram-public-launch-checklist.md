# Instagram Public Launch Checklist

Use this checklist after the KWIN City Instagram account exists and the bio link points to:

```text
https://kwin-city.com/instagram?utm_source=instagram&utm_medium=bio&utm_campaign=social_launch
```

Instagram actions such as pinning posts and creating Story Highlights should be done in the Instagram mobile app.

## 1. Publish the First 3 Pinned Posts

Post these three items first, then pin them to the top of the profile grid.

Instagram allows profile pinning from the mobile app. Open the post, tap the three-dot menu, then choose the pin-to-profile option. Pin no more than three launch posts.

## 3. Verify Public Launch

Do this from another Instagram account or from a logged-out/private browser.

## Public Profile Check

Confirm:

```text
Profile name is KWIN City.
Handle is the final approved handle.
Profile is public.
Bio clearly says what KWIN City is.
Bio link opens kwin-city.com/instagram.
The first three posts are pinned.
Story Highlights appear in the correct order.
```

## Bio Link Check

Tap the bio link and confirm it opens:

```text
https://kwin-city.com/instagram
```

If UTM parameters are present, that is fine:

```text
https://kwin-city.com/instagram?utm_source=instagram&utm_medium=bio&utm_campaign=social_launch
```

## Portal Link Check

On the Instagram hub page, tap:

```text
Read the city brief
Verify the sources

Why North Bengaluru matters

Check the source ledger
What is KWIN City? / Open
Why Doddaballapura? / Open

```

Expected result:

```text
Every tap opens a KWIN City page.
No broken links.
No horizontal scrolling on mobile.
```

## Professional Dashboard Check

Instagram's professional dashboard is available in the mobile app for professional accounts.

In the Instagram app:

```text
Profile -> View professional dashboard
```

Check:

```text
profile visits
website taps
accounts reached
content interactions
```

For launch day, the most important signal is:

```text
website taps from the bio link
```

## 4. Add the Final Instagram URL to the Website

Only do this after the final Instagram handle is confirmed and publicly visible.

Final KWIN City Instagram URL:

```text
https://www.instagram.com/hellokwincityconnect
```

then update `app/layout.tsx`:

```ts
sameAs: [
  'https://www.kiadb.in',
  'https://www.facebook.com/kwincity',
  'https://www.instagram.com/hellokwincityconnect',
],
```

Then run:

```bash
npm run build
```

After deployment, view page source or inspect structured data and confirm the Instagram URL appears in the Organization schema.

## 5. Launch-Day Quality Gate

Before sharing the account broadly, confirm:

```text
Three pinned posts are live.
Eight highlights exist.
Bio link works.
Website hub loads fast on mobile.
Professional dashboard is visible.
Facebook page link is already in the website footer.
No post claims imply government endorsement unless the source proves it.
No caption promises investment returns or unverified certainty.
```
