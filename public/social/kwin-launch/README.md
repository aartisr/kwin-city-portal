# KWIN City Launch Social Artifact

Use this folder for the immediate launch post.

## Instagram

Upload these files as a carousel in this order:

```text
kwin-launch-slide-01.png
kwin-launch-slide-02.png
kwin-launch-slide-03.png
kwin-launch-slide-04.png
kwin-launch-slide-05.png
kwin-launch-slide-06.png
kwin-launch-slide-07.png
kwin-launch-slide-08.png
```

Caption:

```text
instagram-caption.txt
```

Alt text:

```text
Carousel introducing KWIN City as a proposed Knowledge, Wellbeing, and Innovation district in North Bengaluru, with source labels for location, sectors, wellbeing, and evidence.
```

After posting, pin this post to the Instagram profile.

## Upload To Instagram From The Mobile App

1. Open Instagram and switch to `@hellokwincityconnect`.
2. Tap the `+` create button.
3. Choose `Post`.
4. Tap the multiple-select/carousel icon.
5. Select slides `01` through `08` in order.
6. Keep the crop as square.
7. Tap `Next`.
8. Do not add heavy filters; keep the brand colors intact.
9. Paste the text from `instagram-caption.txt`.
10. Add the alt text below under advanced accessibility settings if available.
11. Publish.
12. Open the post, tap the three-dot menu, and pin it to the profile.

## Facebook

Reuse the same carousel images, or use only:

```text
kwin-launch-slide-01.png
```

Caption:

```text
facebook-caption.txt
```

## Upload To Facebook From The Page

1. Open Facebook and switch into the `KWIN City` Page.
2. Start a new post.
3. Choose Photo/Video.
4. Upload all eight carousel images, or upload only `kwin-launch-slide-01.png` if you want a simpler first Facebook post.
5. Paste the text from `facebook-caption.txt`.
6. Confirm the link preview or visible URL points to `kwin-city.com/instagram`.
7. Publish.
8. Pin the post to the top of the Facebook Page if this is the first public launch post.
